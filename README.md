1. **Clone this repository**. Use ChatGPT for help if needed.
2. **Navigate to the cloned folder**.
3. **Activate the virtual environment** by running:
   ```shell
   .\env\Scripts\activate
   ```
4. **Run the project** by executing:
   ```shell
   python mm.py
   ```
